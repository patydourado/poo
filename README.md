Este repositório contém o material didático, exemplos de código e exercícios da disciplina de **[Programação Orientada a Objetos(POO)]**, lecionada nos cursos de Computação.

---

## 📂 Organização do Repositório (Em construção...)
Para manter a organização, os arquivos estão divididos da seguinte forma:

* `/exemplos`: Códigos-fonte demonstrados em sala de aula.
* `/exercicios`: Listas de exercícios e enunciados de trabalhos.
* `/recursos`: Imagens e diagramas utilizados na documentação.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** [Java]
* **IDE Sugerida:** [Eclipse]

## 🚀 Como utilizar este material
Para clonar este repositório e ter acesso aos códigos localmente, utilize o comando:

`git clone https://github.com/patydourado/poo.git`
